# NEXUS CONTACT MANAGER

A modern Contact Management System built using Python and CustomTkinter.

---

## Features

- Add Contacts
- Update Contacts
- Delete Contacts
- Search Contacts
- Modern CRM-style UI
- JSON Data Storage
- Dark Theme Dashboard
- Contact Selection System
- Responsive Layout

---

## Tech Stack

- Python
- CustomTkinter
- JSON

---

## Installation

Install required library:

```bash
pip install customtkinter
```

---

## Run Project

```bash
python app.py
```
