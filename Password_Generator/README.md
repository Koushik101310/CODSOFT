# CYBERPASS PASSWORD GENERATOR

A futuristic Password Generator built using Python and CustomTkinter.

---

## Features

- Generate Secure Passwords
- Password Strength Indicator
- Copy Password Feature
- Password History
- Modern Cybersecurity UI
- Dark Theme Dashboard
- Adjustable Password Length
- Uppercase, Lowercase, Numbers & Symbols Options

---

## Tech Stack

- Python
- CustomTkinter
- JSON

---

## Installation

Install required library:

```bash
pip install customtkinter
```

---

## Run Project

```bash
python app.py
```
